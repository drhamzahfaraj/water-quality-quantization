# Water Quality Quantization

Organized code and supplementary files for edge AI quantization experiments based on the UNEP GEMS/Water dataset.

# Structure

- experiments/ — configs and per-run logs
- models/ — checkpoints for teacher, student, ablations
- src/ — code modules for training, analysis
- supplementary/ — regional metrics, error histograms, ablations
